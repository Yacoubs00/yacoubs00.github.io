Welcome to Yacoub's Subtitle Addon Repository for Kodi

Installation

Download Repository

Quick Install:

  1. In Kodi: Settings → File Manager → Add source
  2. Enter: https://yacoubs00.github.io/
  3. Name it: Yacoub Repo
  4. Go to: Settings → Add-ons → Install from zip file
  5. Select "Yacoub Repo" and install repository.yacoubs.subs-1.0.0.zip
  6. Install addons from: Install from repository → Yacoub's Subs Repository
